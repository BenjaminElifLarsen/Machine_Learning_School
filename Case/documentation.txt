Case:
Identifikation af fugle. Visse fuglearter kan være svære at artsbestemme fra andre, samt nogle gange kan man ikke se fuglen.
Datasæt:
Alle features burde kunne bruges, da datasætttet er baseret ud fra et andet.
Selve datasættet består af features fra to grupper, Chromagrammer og Spectral Centroider.
Algorithmen:
Syv forskellige blev brugt for at se hvilken der var gode eller dårlige.
De er alle sammen supervised.
Et af dem er Deep Learning.
Nogle af dem var tæt på hinanden, f.eks. Linear SVC og SVC linear kernel. Andre som Neural Network og Decision Tree Classifier er meget forskellige fra resten.
	K-Nearest Neighbour
	Linear SGD
	Linear SVC
	SVC Linear Kernel
	Gaussian Naive Bayes
	Decision Tree Classifier
	Multi-Layer Perceptron Neural Network
Konklusion: 
Problemet kunne nok ikke løses uden Maskine Læring. 
Modelleren kunne nok forbedres via mere data i det balanceret træningssæt.
En fugleart var ikke til stede i testsættet, samt visse fuglearter kunne har været bedre repræsenteret.
F.eks. en fugleart kunne være svære at identifere, men for få til at det kunne blive opdaget i testsættet.
